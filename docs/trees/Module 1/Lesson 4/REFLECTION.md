Let’s look at the tree:

        1
       / \
      2   3
     / \
    4   5

To find the parent of 5, follow the direct connection:

5 ← child of 2

So:

Parent(5) = 2


2. Is node 3 a leaf?

Node 3 has:
- no left child
- no right child

So:

✔ Yes — node 3 is a leaf

Because:

Leaf = node with no children

Even though it is directly attached to root, that doesn’t matter.

3. Subtree of node 2

A subtree is:

The node itself + everything below it

So:

    2
   / \
  4   5

That is the entire subtree rooted at 2.
